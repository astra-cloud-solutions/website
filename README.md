# Astra Cloud Solutions — Static Website

Simple landing site for Astra Cloud Solutions. a consulting firm focused on AWS migration, cloud architecture, and DevOps automation.

## Contents
- index.html — main site
- styles.css — site styles
- LICENSE — MIT license (branding/logo are proprietary)
- README.md — this file

## Preview locally
1. Open a terminal in the project folder.
2. Start a simple server:
```sh
python3 -m http.server 8000
```
3. Open in your browser:
http://localhost:8000/index.html

## Develop
- Edit HTML/CSS files directly.
- Keep assets (favicon, images) in the project root or an `assets/` folder.
- Use modern browsers for development and testing.

## Deployment
This is a static site and can be deployed to GitHub Pages, Netlify, Vercel, or any static hosting provider.
